## 🚀 **DEPLOYMENT READY**

### Prerequisites
- Node.js 20+
- npm 10+
- Docker (optional)

### Build Commands
```bash
# Install dependencies
npm install

# Run tests
npm test
npm run test:coverage
npm run test:e2e

# Build for production
npm run build

# Preview production build
npm run preview

# Analyze bundle
npm run analyze
```

```bash

VITE_WS_URL=ws://localhost:3001
VITE_SENTRY_DSN=your-sentry-dsn
```
