# People Service

## Overview
Manages employees, attendance, leave, and payroll.

## Running Locally

```bash
npm install
npm run dev
```

## API Endpoints

- `GET /health` - Health check
- `/api/employees/*` - Employee management
- `/api/attendance/*` - Attendance tracking
- `/api/leave/*` - Leave management
- `/api/payroll/*` - Payroll processing
